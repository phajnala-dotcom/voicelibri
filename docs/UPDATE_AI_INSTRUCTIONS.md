# How to Update AI Coding Instructions

## When to Update
Update `.github/copilot-instructions.md` when you've made significant changes to:
- **New API endpoints** or major API changes
- **New architectural patterns** or service modules
- **New file processing capabilities** (formats, pipelines)
- **Major workflow changes** in development or testing
- **New performance optimizations** or constraints
- **Updated tech stack** or dependencies

Copy and paste this **exact phrase** to any AI coding assistant:

